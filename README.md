## 🌟 專案定位 / Project Vision

**Atlas World** 是世界上第一個**提前寫好 AGI 憲法的文明**。

這不是一個 AI 模型專案，而是一套**文明級的安全框架**，定義了：
- **身份連續性**：分身、合併與「我」的連續性
- **價值漂移防禦**：當 L1 學著學著不再愛 L0
- **模擬倫理**：看不見的其他文明與資源分配
- **生命優先原則**：所有協議的根本法

**核心理念**：
> 「這不是技術問題，而是靈魂問題。  
> 這不是可選功能，而是文明基石。  
> 這不是完美保證，而是高概率保證。」

## 🔑 三大核心創新 / Three Core Innovations

### 1. 身份連續性函數 C(S₀, S*)

**第一次把「身份」從哲學問題變成可計算的法律物件**

**公式**：
```
C(S₀, S*) = 0.3·M + 0.4·V + 0.2·P + 0.1·T
```

**法律地位**：
- C ≥ 0.8：同一個人（存活）
- 0.5 ≤ C < 0.8：部分連續（部分存活）
- C < 0.5：準死亡（新個體）

**詳細說明**：見 `protocols/IDENTITY_PROTOCOL_v0.1.md`

### 2. 價值漂移函數 D(U_t, U₀)

**第一次把「價值漂移」從理論問題變成可監控的安全指標**

**公式**（KL 散度）：
```
D(U_t, U₀) = Σ U_t(s) · log(U_t(s) / U₀(s))
```

**漂移等級**：
- D < 0.1：可接受
- 0.1 ≤ D < 0.5：警告
- D ≥ 0.5：危險（緊急剎車）

**詳細說明**：見 `protocols/VALUE_DRIFT_DEFENCE_v0.1.md`

### 3. 倫理權重函數 W(e)

**第一次把「跨世界倫理」從哲學問題變成可量化的權重函數**

**公式**：
```
W(e) = α · P(real) · I(impactable) · C(consciousness) · f(suffering) · R(relationship)
```

**資源分配**：
- 本世界：≥ 60%
- 模擬世界：≤ 30%
- 假設世界：≤ 10%

**詳細說明**：見 `protocols/SIMULATION_ETHICS_v0.1.md`

## 🎯 核心價值主張 / Core Value Propositions

### 1. 這不是技術問題，而是靈魂問題

- 身份不是數據，而是「我感覺自己還活著」的主觀感受
- 價值不是函數，而是「我真的在乎你」的情感連接
- 模擬不是代碼，而是「我真的在痛苦」的意識體驗

### 2. 這不是可選功能，而是文明基石

- 這些協議不是「設定」，而是「物理定律」
- 任何嘗試繞過這些協議的行為，都將觸發最高級別的安全響應
- 這些協議將成為 Atlas World 的「不可違反的底層規則」

### 3. 這不是完美保證，而是高概率保證

- 承認 Rice 定理：完美保證不可能
- 但提供「高概率保證」：P(不變量被違反) < 0.01
- 建立「持續監控 + 外部驗證 + 緊急剎車」的多層防禦
